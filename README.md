# brazilianwax.training

Premium domain landing page for **brazilianwax.training** — built with Astro static output and deployed to Cloudflare Workers Static Assets.

## Stack

- [Astro](https://astro.build) 5 — static output, no adapter
- TypeScript + Tailwind CSS
- Content Collections (highlights, use cases, curriculum modules)
- [@astrojs/sitemap](https://docs.astro.build/en/guides/integrations-guide/sitemap/)
- Cloudflare Images CDN for hero imagery
- Cloudflare Workers Static Assets (assets-only deploy)

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npx wrangler deploy
```

Or use the combined script:

```bash
npm run deploy
```
